# Personal Expense Tracker

A website to track income and expenses. Sign up, log in, add your transactions, and see your totals on a dashboard.

**Live demo:** https://expense-tracker-production-8dad.up.railway.app

## What it does

- Sign up and log in (each user only sees their own data)
- Add, edit, and delete income/expense entries
- See total income, total expenses, and balance
- Search and filter transactions

## Built with

- Python (Flask) — backend
- MySQL — database
- HTML/CSS (Bootstrap) — frontend
